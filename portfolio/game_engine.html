A colleague and I developed a simple 2D Game Engine in C++ using the libraries SDL2, Glad (for OpenGL and GLSL shaders), OpenGL Mathematics (GLM), Box2D (for the physics), OpenAL and AudioFile.

We also developed a simple 2D space shooter in this engine using the assets from the game Xenon 2000.

This engine and game were delivered as projects for two different classes: Advanced Game Programmign Topics and Computer Graphics. We were graded 19.2 out of 20 and 19.9 out of 20 for each class respectively.
